# Lab 2 Solution Summary

## Dataset
- Name: Iris Dataset (UCI Machine Learning Repository)
- Source: https://archive.ics.uci.edu/dataset/53/iris
- Raw file used: https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data
- Type: Tabular dataset with numeric input features and a categorical class label
- Size: 150 rows, 4 feature columns, 1 target column

## Machine Learning Problem
This is a **classification** problem.
Given flower measurements (sepal length, sepal width, petal length, petal width), the model is expected to predict the iris species class.

## Target Variable
- `species` (`Iris-setosa`, `Iris-versicolor`, `Iris-virginica`)

## Problem Statement
Use supervised machine learning to classify iris flowers into one of three species based on four numerical flower measurements.

## Why this dataset is suitable
- Open and widely used academic benchmark dataset
- Clear supervised setup with an explicit target label
- Clean tabular structure suitable for Pandas and introductory ML workflows
